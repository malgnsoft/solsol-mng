### ✅ 방금 완료 (코드만 — 미검증·미배포)

- **미커밋** — `useRuntimeConfig(event)` 전환분이 working tree만(배포됨·미커밋) → origin main 커밋 필요.

### ⛔ 운영자 시드·계정 위생 (담당자 자기 env)
- **운영자 시드 미적용** — `seed.admin.FINAL.sql`(실 PBKDF2) 준비 완료, DBA 미적용 → `superadmin` 로그인 401.

- **`/ops/migrate`·`/ops/seed-admin` 라우트 잔존** — 운영 데이터 유입 후 제거 권장(OPS_SECRET 게이트됨).

### 🟡 시크릿·인프라
- **TOSS 미주입**(staging-gate) · **CF WAF 미설정** · **`OPS_SECRET` 배포 중 재설정**(값 미보관, 다음 `/ops` 시 재설정) · **`ALLOWED_ORIGINS`(프론트 CORS) 미설정** · **R2 미바인딩**(문의 첨부).

### 🟡 staging-gate 실연동 (결제 라운드 GO 조건)
- 결제·구독·카드등록·프로비저닝·웹훅 = 골격 + TODO.
- **토스 웹훅 서명검증 완성** — HMAC 골격만(헤더·canonicalization TODO), 완성 전 상태전이 활성 금지.
- **refresh 세션원장 부재** — stateless 서명 → 로그아웃 후 TTL(14일) 내 재발급 가능, 재사용 탐지 없음 → **`TB_SESSION` 도입 필요**(refresh 쿠키 `SameSite=None`).

### 🔴 정책 미확정 (오너/기획)
- **사이트 프로비저닝 방식** — 사이트 생성 시 테넌트 스키마 생성이 동기(brand-api 직접) vs 비동기(배치/워커). 현재 `TB_SITE`·`PROVISION_LOG` 기록까지만.
- **구독 정책** — 플랜변경 비례정산 계산식 · 연장 할인율 · 카드삭제 시 활성구독 재바인딩 · 브랜드 탈퇴 시 개인정보 파기 vs 법정보존(거래기록) 분리.
- **`subjectType`·`payment.method` 코드표** · 전용 **멱등 테이블(uk)** 신설 여부(현재 활성행 프록시 → 동시요청 레이스 잔존).
