## ✅ 현재까지 완료 (참고)
- 관리자단 **107화면 전 도메인 구축**(목데이터) → 공개 미리보기 배포.
- **실 백엔드 연동**: BFF(Nitro `server/`·httpOnly 세션·프록시 화이트리스트)·실 로그인(`admin@solsol.dev`)·`/api/admin/me`.
- **조회형 전 도메인** 실 API 전환(대시보드·사용자·상품·콘텐츠·판매·마케팅·운영·사이트·정산·설정·통계, 어댑터 9종·목 폴백).
- **쓰기(CRUD) 착수**: CSRF 방어 쓰기 BFF 핸들러 + **게시판 CRUD 실 연동**(생성/수정/삭제 E2E). 에러봉투 정규화·낙관적 갱신·CSRF/SSRF 가드.

## 🟦 D. SoT 소유자 확정 대기 (에스컬레이션 — 기획/DBA)

1. **강사 `operation`(운영) 권한 불일치** — 06 §2.8(차단) vs 05 M-2 표(허용). 코드는 상위권위 05를 따름. 06 정렬 필요.
2. **자막 편집 화면(S-AD01-0301-005) 유효성** — 데이터모델 자막 테이블 삭제(위캔디오 위임)와 상충 → 위캔디오 프록시 재정의 vs 제거 결정.
3. **폴더 삭제 시 콘텐츠 처리 정책** — 미분류 이동 vs 하위 함께 삭제.

## ⚙️ E. 배포·운영 위생

- **read-after-write ~60s 지연**(Hyperdrive 쿼리 캐시) — 낙관적 갱신으로 완화, 근본은 인프라 특성.
- 데모 종료 시 `NUXT_PREVIEW_DEMO=false`(공개 원클릭 데모 차단).
- **`SEED_ADMIN_PASSWORD` Pages secret화·로테이션**(현재 서버 runtimeConfig 기본값 번들 — 클라 미노출이나 운영 위생상 이전 권장).
- `devLoginEmail` public prefill 정리(무해하나 프로덕션 제거).
- **공개 원클릭 데모 = 실 owner 접근**(dev 데이터·읽기 프록시·쓰기 게시판만) — 노출 범위 정책 결정. 필요 시 Cloudflare Access 게이트.

## 🧱 F. 기술 부채

- `vue-tsc` 미설치(`nuxt.config` `typeCheck:false`) → 타입 게이트 미커버. baseline 타입오류 ~211건(공용 `AppDataTable`의 `Record<string, unknown>` 계약 부채) — 별도 리팩터 트랙.
